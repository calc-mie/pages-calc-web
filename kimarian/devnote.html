開発ノート

プロジェクトを整理するための情報を乗っけとくページ。

使用ライブラリ

Mirror

概要 : TCPで動く安定したUNETみたいなやつ。APIがほぼUNETだからUNETのドキュメントがMirrorのドキュメント

使い所さん : ネットワークの管理のほぼすべて。

UniRx

概要 : イベント処理周りをかきやすくしてくれる。

使い所さん : オブジェクトが死んだ時に起爆したい処理をするみたいなやつを書く時に使う。ガッチガチにこのプログラミングスタイルで開発する頭がないので、利用できそうなところだけする。

主なクラス

Entity周り

Entity

概要 : ワールドにいるオブジェクトとかにつける。

publicメンバ

  • Id : string - オブジェクトの名前。(正直Idという命名がやばいのでそのうちIdとNameとかで分けるつもり)
  • InteractManager : InteractManager - オブジェクトに登録されたコマンド(右クリックするとでるやつ)の取得/実行を制御するオブジェクト
  • CanInteract : bool - コマンドを実行できるオブジェクトかどうかを示す
  • SetId(string id) : void - Idを変更する時にこのメソッドを呼ぶ

protectedメンバについて : Startがprotected virtualになってるので、初期化処理を書き換えたい時はoverrideして

Serialized Field

  • defaultId - 変更せずとも設定されるId
  • textOffset - オブジェクトの上に出るIdのテキストの位置
  • textColor - 同上の色

Character : Entity

概要 : プレイヤーや敵モブなどのステータス持ちEntityに継承される。

publicメンバ

  • Status : Status - ステータスを管理するオブジェクト

PlayerSystem : Character

概要 : Playerのゲームオブジェクトにアタッチする。

Collectable : Entity

概要 : 拾えるゲームオブジェクトにアタッチする。

SimpleNPC : Character

概要 : 喋れるゲームオブジェクトにアタッチする。
